# Contributing

Thanks for helping improve Luminux! To keep contributions consistent and safe, please follow these steps:

1. Fork the repository and create a feature branch.
2. Install dependencies with `npm install`.
3. Run `npm run lint` and `npm run typecheck` to keep the codebase consistent.
4. Run `npm run build` to ensure production builds remain stable.
5. Submit a pull request describing the change and why it avoids breaking existing flows.

## Coding notes
- Keep changes minimal and aligned with existing patterns.
- Update `src/lib/site.ts` when altering branding, navigation, or pricing.
- Add or update tests/docs when introducing new functionality.
